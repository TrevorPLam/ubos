# Agreements API

## Implemented
- GET /api/proposals
- POST /api/proposals
- PATCH /api/proposals/:id
- POST /api/proposals/:id/send
- DELETE /api/proposals/:id

- GET /api/contracts
- POST /api/contracts
- PATCH /api/contracts/:id
- POST /api/contracts/:id/send
- POST /api/contracts/:id/sign
- DELETE /api/contracts/:id

## Gaps vs plan
- Template/versioning, e-sign provider integrations, signature packets.
